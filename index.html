About
Me

I have a diverse skillset that spans the entire spectrum of modern web technologies. In the front end, a solid foundation in HTML and CSS enables me to craft beautiful and responsive user interfaces. Leaning on my experience in JavaScript, I elevate those interfaces and applications by adding interactive and dynamic features that keep users engaged.

Not just limited to the front end, I have a strong backend development background as well. I've build web applications using Python's Flask and Django full-stack frameworks, focusing on both functionality and aesthetics. Additionally, I am currently in the process of building an application that utilizes a series of microservices I'm developing using Java's Dropwizard framework. In each of these projects, I've also built, connected to, and performed queries and updates to postgreSQL databases.
